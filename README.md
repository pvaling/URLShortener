# URL Shortener

For coding practice

## Features

- Pluggable storage backends
    - InMemory
    - SQLBackend
      - SQLite included
- Collision handling with tests and retries
- Basic tests for TDD approach
- PyCharm HTTP client examples
- Written with Fast API and SQLAlchemy
- Object-oriented design of Shortener
